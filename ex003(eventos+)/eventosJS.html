Eventos JS

Eventos de mouse

mouseenter -> quando o mouse entra na DIV

mousemove -> quando o mouse se move pela DIV

mousedown -> quando é segurado o clique na DIV

mouseup -> quando é soltado o clique na DIV

click -> quando é feito o clique sobre a DIV

Funções

É um conjunto de códigos que vai ser executado quando um evento ocorres

esse conjunto é chamado de BLOCO

a ação deve ser indicada no documentO html, como por ex: div onclick="nomedafunção()" -> isso executa no evento click algo

Pode-se ainda, adicionar o evente com o addEventListener('evento', nomedoevento), ex : var.addEventListener('click', clicar)
