Variaveis

    tipos primitivos :
  • 1. Number
    • 1.1 Number.parseInt ou somente parseInt
    • 1.2 Number.parseFloat ou somente parseFloat
  • 2. String

Artifícios

  • window,alert() ---> exibe o que estiver dentro do parenteses numa janela de alerta
  • window.prompt() ---> pede ao usuario que digite algo
  • document.write() ---> escreve no documento html o que estiver dentro dos parenteses
  • `O texto é digitado aqui e em seguinda a sua ${variável} ` ---> modelo de texto com máscara de variável

    • Manipulação de texto

    • .length ---> descreve o numero de caracteres de uma string
    • .toUpperCase() ---> transforma o texto em maiusculo
    • .toLowerCase() ---> transforma o texto em minusculo

    Manipulação de números

    • .toFixed(2) ---> No exemplo limitaria um float para 2 casas decicais após o ponto
    • .replace('.', ',') ---> no exemplo troca , por .
    • .toLocaaleString('pt-BR', {style:'currency', currency:'BRL'}) ---> transforma o numero para a moeda monetaria brasileira, o R$. Pode se ainda usar para dolar 'USD' e euro 'EUR'

Simbolos

  • ! ---> negação (!true == falso // !false == verdadeiro)
  • && ---> significa 'e'
  • || ---> significa 'ou'
